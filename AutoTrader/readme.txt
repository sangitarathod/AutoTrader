== Description ==

AutoTrader is a blank canvas for your ideas and it makes the block editor your best brush.

With new block patterns, which allow you to create a beautiful layout in a matter of seconds.

== Manual Installation ==

Must have Autotrader.zip 
Go to Admin panel Appearance->Addnew->uploads->select Autotrader.zip->Activate
